アロー関数

アロー関数は簡単に言えば　function(){} という書き方を短縮したもの(ショートハンド)である。 


        const sayHello! = function(name){
            console.log("Hello" + name + "!");
        }

        sayHello('tanaka');

        const sayGoodBye! = (name) => {
            console.log("GoodBye" + name + "!");
        }

        sayGoodBye('tanaka');

厳密には、functionとアロー関数の書き方で、クラスにおけるthisの挙動が変わるが、クラスを書く機会がない限り、その違いは覚えなくて良い。モダン開発においては、functionの書き方よりも見やすいので、アロー関数がよく使われる。 


        const button = document.getElementById('button');

        button.onclick = () => {
            alert("Hello!");
        }
